# Supabase MCP Data Assistant

## 1. Project Description
Un tableau de bord moderne en mode sombre pour un assistant de données IA appelé "Supabase MCP Data Assistant". L'application présente une interface de chat à deux colonnes permettant d'interroger des bases de données via le Model Context Protocol (MCP). L'objectif est de fournir une interface SaaS élégante pour visualiser et interagir avec des données logistiques et des PV d'établissements.

## 2. Page Structure
- `/` - Tableau de bord principal (Home)

## 3. Core Features
- [ ] Interface de chat avec messages utilisateur et réponses IA
- [ ] Sidebar avec navigation et informations MCP
- [ ] Affichage de tableaux de données structurés
- [ ] Graphiques à barres horizontales pour visualisation des données
- [ ] Barre d'entrée de texte pour poser des questions
